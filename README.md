## Chat-Ex
用 vue.js + socket.io 所製作，很簡單的聊天室範例

## 用法
(1)安裝相關套件：
npm install

(2)啟動 socket server：
node server.js

(3)用瀏覽器連線：
http://127.0.0.1:8080
